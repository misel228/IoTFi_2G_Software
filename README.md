# IoTFi_2G_Software

IoT-Fi is a compact low power IoT enabled device having the capability of GSM module (SIM868) for providing network connectivity via using sim card and having the power of RP2040 chip. ## Features:
* ESP-32(C3)
* USB type-C for power-up board and uploading codes in rp2040
* Five user Buttons
* SD card slot for storing data
* On Board RP2040
* Two antenna connectors
* A 3.5mm jack for audio input output
* 1.14 LCD display
